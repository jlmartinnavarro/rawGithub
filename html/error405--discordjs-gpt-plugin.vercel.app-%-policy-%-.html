Privacy Policy for Discord.js docs navigator

Information Collection

The tool does not collect, store, or process any personal information from its users. Everything is sent anonymously, the only information we get is the query to search for in the documentation.

Data Processing and Storage

The backend code of the tool processes search queries in real-time and does not store these queries or any associated data. The tool operates using the following services:

  • FastAPI for server-side operations.
  • Jaro-Winkler distance metric for the search algorythm.
  • Requests library for fetching the documentation.

Security

No personal data is collected or stored, minimizing the chances of a security breach.
